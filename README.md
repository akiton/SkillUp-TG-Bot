# Команда Пюрешка

## Технологии
- Docker
- Docker Compose
- Python 3.12
- Pydantic
- Aiogram
- PostgreSQL

## Настройка
1. Скопируйте `.env.example`. 
2. Переименуйте их в `.env`.
3. Заполните конфигурационные файлы.

## Запуск приложения
```bash
  docker-compose up --build
```
